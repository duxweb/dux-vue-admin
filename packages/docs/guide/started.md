# 快速开始

## 环境准备

### 环境要求

- Git 版本2.25或更高。
- Node.js 版本20或更高。
- Bun 用于方便的依赖管理。

### 开发工具

我们推荐使用 VS Code 作为开发工具，并可以安装以下插件来提升开发体验：

- ESLint
- UnoCSS

## 本地部署


### 1. 初始化项目

此项目提供了内置的脚手架安装方法，强烈推荐使用这种方式来初始化项目。

```bash
npx @duxweb/dux-vue-admin-cli init
```

### 2. 安装依赖

项目中执行以下命令安装依赖

```bash
bun install
```

### 3. 启动项目

```bash
bun dev
```
